pwd - Mostra o local (diretório) do qual você se encontra atualmente. ls - lista o conteúdo dentro da pasta/diretório. ls -a - mostra todos os arquivos (ocultos) da pasta/diretório. ls -al (ll) - "longa lista", mostra mais detalhes da lista. ls --help - mostra os demais comandos que podem ser usados. man ls - também ctrl +l e clear: limpa a tela cd - change director, muda de diretório cd /etc - etc é onde fica as configs do sistema . - diretório corrente .. - diretório "pai" do qual você está. cd - - volta para o diretório anterior mkdir - make a directory - cria um diretório: por exemplo, utiliza-se: mkdir teste touch - cria arquivos zerados rmdi - remove diretório. cp - copy (copiar diretório) - r - conteúido dentro de um diretório exempolo de copy: cd dir1 cp -r * ../dir2 ls para renomear: mv dir1 dir4 (renomeou dir1 para o dir4) history - mostra todos os comandos já utilizados. ls * - mostra tudo menoss ocultos. outros exemplos: ls arq1* retorna arq1, arq10 e arq100 ls arq1 retorna arq10 se eu possuo um arquivo, do qual existem outros numerados de 1 até 100, mas eu preciso listar os de 1 até 5, eu faço dessa maneira (porém os ? são de acordo com o tamanhode caracter de um arquivo. por exemplo, "arq5" ls ???[1-5]
